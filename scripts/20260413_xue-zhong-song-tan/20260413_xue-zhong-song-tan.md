## ③ 字义分析
### 【雪 · xuě】
**中文**
```
「雪」，就是雪，下雪的雪。
```
**拼音**
```
「Xuě」, jiù shì xuě, xià xuě de xuě.
```
**영어**
```
「雪」 means snow.
```

**中文**
```
比如：下雪、雪花、大雪。
```
**拼音**
```
bǐrú: xià xuě, xuě huā, dà xuě.
```
**영어**
```
For example: 下雪 (to snow), 雪花 (snowflake), 大雪 (heavy snow).
```

**中文**
```
在这个成语里，雪代表一种寒冷的、困难的处境。
```
**拼音**
```
zài zhège chéngyǔ lǐ, xuě dàibiǎo yī zhǒng hánlěng de、kùnnán de chǔjìng.
```
**영어**
```
In this idiom, 雪 represents a cold, difficult situation.
```

### 【中 · zhōng】
**中文**
```
「中」，表示在某种状态或处境之中。
```
**拼音**
```
「Zhōng」, biǎoshì zài mǒu zhǒng zhuàngtài huò chǔjìng zhī zhōng.
```
**영어**
```
「中」 means being in the middle of a certain state or situation.
```

**中文**
```
比如：途中、困境中、黑暗中。
```
**拼音**
```
bǐrú: tú zhōng, kùnjìng zhōng, hēi'àn zhōng.
```
**영어**
```
For example: 途中 (on the way), 困境中 (in difficulty), 黑暗中 (in the dark).
```

**中文**
```
雪中，就是在大雪之中，也就是在最艰难的时候。
```
**拼音**
```
xuě zhōng, jiù shì zài dà xuě zhī zhōng, yě jiù shì zài zuì jiānnán de shíhou.
```
**영어**
```
雪中 means being in the snow — in other words, during the most difficult times.
```

### 【送 · sòng】
**中文**
```
「送」，是给予、赠送的意思。
```
**拼音**
```
「Sòng」, shì jǐyǔ、zèngsòng de yìsi.
```
**영어**
```
「送」 means to give or deliver something.
```

**中文**
```
比如：送礼、赠送、送行。
```
**拼音**
```
bǐrú: sòng lǐ, zèngsòng, sòng xíng.
```
**영어**
```
For example: 送礼 (give a gift), 赠送 (donate/give), 送行 (see someone off on a journey).
```

**中文**
```
这里的送，带有主动帮助别人的温暖感。
```
**拼音**
```
zhèlǐ de sòng, dài yǒu zhǔdòng bāngzhù biérén de wēnnuǎn gǎn.
```
**영어**
```
The 送 here carries a warmth of actively reaching out to help someone.
```

### 【炭 · tàn】
**中文**
```
「炭」，是炭火的炭，古代用来取暖的燃料。
```
**拼音**
```
「Tàn」, shì tàn huǒ de tàn, gǔdài yòng lái qǔ nuǎn de ránliào.
```
**영어**
```
「炭」 is charcoal — the fuel used for heating in ancient times.
```

**中文**
```
比如：炭火、木炭、炭盆。
```
**拼音**
```
bǐrú: tàn huǒ, mùtàn, tàn pén.
```
**영어**
```
For example: 炭火 (charcoal fire), 木炭 (charcoal), 炭盆 (charcoal brazier).
```

**中文**
```
在这个成语里，炭代表温暖，代表你在最需要的时候得到的帮助。
```
**拼音**
```
zài zhège chéngyǔ lǐ, tàn dàibiǎo wēnnuǎn, dàibiǎo nǐ zài zuì xūyào de shíhou dédào de bāngzhù.
```
**영어**
```
In this idiom, charcoal represents warmth — the help you receive when you need it most.
```

## ④ 成语整体含义
**中文**
```
「雪中」——在大雪之中，代表最困难的处境；「送炭」——送去炭火，代表给予温暖和帮助。
```
**拼音**
```
「Xuě zhōng」—— zài dà xuě zhī zhōng, dàibiǎo zuì kùnnán de chǔjìng；「sòng tàn」—— sòng qù tàn huǒ, dàibiǎo jǐyǔ wēnnuǎn hé bāngzhù.
```
**영어**
```
「雪中」 — being in heavy snow, representing the most difficult situation; 「送炭」 — sending charcoal, representing giving warmth and help.
```

**中文**
```
字面意思是：在下雪天给人送来炭火取暖。
```
**拼音**
```
zìmiàn yìsi shì: zài xià xuě tiān gěi rén sòng lái tàn huǒ qǔ nuǎn.
```
**영어**
```
Literal meaning: sending charcoal to keep someone warm on a snowy day.
```

**中文**
```
引申义是：在别人最困难、最需要帮助的时候，给予及时的帮助。
```
**拼音**
```
yǐnshēn yì shì: zài biérén zuì kùnnán、zuì xūyào bāngzhù de shíhou, jǐyǔ jíshí de bāngzhù.
```
**영어**
```
Extended meaning: giving timely help when someone is at their most difficult moment and most in need.
```

**中文**
```
它的反义词是「锦上添花」——在本来就很好的情况下再添一分美好，而不是在困难中给予帮助。
```
**拼音**
```
tā de fǎnyìcí shì「jǐn shàng tiān huā」—— zài běnlái jiù hěn hǎo de qíngkuàng xià zài tiān yī fēn měihǎo, ér bú shì zài kùnnán zhōng jǐyǔ bāngzhù.
```
**영어**
```
Its antonym is 「锦上添花」 — adding flowers to brocade, meaning to enhance something already good, not helping someone in need.
```

**中文**
```
近义词有：急人之难、及时雨。
```
**拼音**
```
jìnyìcí yǒu: jí rén zhī nán, jíshí yǔ.
```
**영어**
```
Near synonyms include: 急人之难 (rushing to help in times of need) and 及时雨 (timely rain — a person who helps just in time).
```

## ⑦ 使用注意
**中文**
```
「雪中送炭」是一个褒义词，用来称赞别人在困难时刻给予帮助。
```
**拼音**
```
「Xuě zhōng sòng tàn」shì yī gè bāoyìcí, yòng lái chēngzàn biérén zài kùnnán shíkè jǐyǔ bāngzhù.
```
**영어**
```
「雪中送炭」 is a positive expression used to praise someone for helping in difficult moments.
```

**中文**
```
注意：这个成语强调「时机」——只有在别人真正困难的时候才能用，如果对方本来就过得不错，这个成语就用不上了。
```
**拼音**
```
zhùyì: zhège chéngyǔ qiángdiào「shíjī」—— zhǐyǒu zài biérén zhēnzhèng kùnnán de shíhou cái néng yòng, rúguǒ duìfāng běnlái jiù guò de búcuò, zhège chéngyǔ jiù yòng bu shàng le.
```
**영어**
```
Note: this idiom emphasizes 「timing」— it can only be used when someone is truly in difficulty. If the person is already doing well, this idiom doesn't apply.
```

**中文**
```
还有一点：「雪中送炭」和「锦上添花」是反义词，大家要区分清楚。前者是困难中的帮助，后者是顺境中的额外美好。
```
**拼音**
```
hái yǒu yīdiǎn:「xuě zhōng sòng tàn」hé「jǐn shàng tiān huā」shì fǎnyìcí, dàjiā yào qūfēn qīngchǔ. Qiánzhě shì kùnnán zhōng de bāngzhù, hòuzhě shì shùnjìng zhōng de éwài měihǎo.
```
**영어**
```
One more thing: 「雪中送炭」 and 「锦上添花」 are antonyms — make sure to distinguish them. The former is help in hardship; the latter is an extra blessing in good times.
```
