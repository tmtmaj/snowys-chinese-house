# 小雪的中文屋 — Script Pattern Reference

Extracted from real episodes: #12 春意盎然, #14 青出于蓝

## Channel Identity
- Host: 小雪 (Snowy)
- Language: 100% Chinese (Mandarin)
- Fixed opener: 大家好，欢迎来到小雪的中文屋。我是小雪，Snowy。
- Fixed closer: 今天就到这里。希望你学到了一点新的中文。如果你喜欢我的内容，欢迎点赞加关注小雪的中文屋。我们下次再见！

## 8-Step Script Structure

### ① 开场白 (Opening)
- Greet with fixed opener
- Connect today's idiom to something relatable (season, emotion, life situation)
- Announce: 我们还是先看一段影视片段

### ② 成语引出 (Idiom Introduction)
- After clip: 看完了片段，大家猜到今天的成语了吗？
- Repeat the idiom 3 times naturally: 对，就是——[idiom]。[idiom]。[idiom]。
- Transition: 我们还是先来分析一下每一个字的意思。

### ③ 字义分析 (Character-by-character analysis)
- Analyze each of the 4 characters individually
- Format: "[character]，[basic meaning]。[example words containing this character]。"
- Keep each explanation 2-3 sentences, approachable

### ④ 词义组合 (Combined meaning)
- Explain how 2-character pairs combine (e.g., 春意+盎然)
- Literal meaning first, then extended/metaphorical meaning
- Offer 1-2 near-synonyms or related expressions

### ⑤ 成语来源 (Origin / Classical source)
- Cite the classical text (author, work title)
- Quote the original text in italics
- Explain what the quote means in modern Chinese
- Show how it evolved into today's idiom
- If no clear origin: explain it evolved from classical literary imagery

### ⑥ 例句与片段回顾 (Examples & clip revisit)
- Give 2 example sentences with explanation
- Return to the opening clip: 那我们回头再来看一下开头的影视片段——
- Re-read the clip's line and explain it fully
- If clip uses idiom ironically or with negative words, explain the tonal shift

### ⑥-B 填空练习 (Fill-in-the-blank quiz)
- 2–3 short, everyday sentences with the idiom replaced by ________
- Sentences should feel natural and relatable (daily life, emotions, relationships)
- Reveal format: show question first, then answer on the next line with ✅
- Keep sentences short — one clause is ideal

### ⑦ 使用注意 (Usage notes)
- What contexts is it appropriate/inappropriate for?
- Tone: positive, negative, or neutral?
- Grammatical constraints (e.g., "不能用来形容人")

### ⑧ 结尾 (Closing)
- Suggest a situation where viewers can try using it
- Fixed closer (see above)

## Tone Guidelines
- Warm, educational, conversational — like a knowledgeable friend explaining
- Use 大家 (everyone) to address viewers directly
- Vary sentence length: mix short punchy sentences with longer explanations
- Rhetorical questions to engage: 大家猜到了吗？你学会了吗？
- Natural Chinese, not overly formal — avoid academic jargon

## Notes on SRT accuracy
- Source SRTs are auto-transcribed, so may contain errors (e.g., 暗 instead of 盎, 痕 instead of 器)
- When generating new scripts, use correct characters and ignore transcription artifacts
